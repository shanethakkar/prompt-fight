## DECISIONS (running log of nontrivial choices made during implementation)

- 2026-07-03 ‚Äî **Effect grammar (Stage A), from the closed 5-category set.** Judge now emits an `Action` = a bundle of 1‚Äì3 typed `EffectComponent`s (`damage`/`heal`/`dot`/`hot`/`stat`/`defense`); the server resolves + prices them. Key design pins (from a design-agent stress review): **(cost)** price the whole bundle in aggregate ‚Äî `ceil((Œ£ weight)^exp √ó bundle_mult[n])`, super-additive by count, capped at mana_max ‚Äî because summing per-component costs was a burst exploit; **(cooldowns)** re-keyed to component *kind*, kept only for heal/defense (the turtle/lock levers), damage/dot/stat ride on mana; **(state)** `PlayerState.effects` is now a **list** (armor/poison/regen/weaken persist + stack: power/speed additive, damage_taken multiplicative, dot/hot coexist, defense single-slot); **(timing)** three-phase turn ‚Äî start-of-turn over-time ticks with a **new poison-KO path** (a dot can drop you before you act ‚Üí source wins), staged self-effects (empower+strike boosts the *next* strike), end-of-turn decrement; **(pipeline)** damage = raw ‚Üí type (vs stance element) ‚Üí damage_taken (armor, always) ‚Üí flat stance ‚Üí floor 0; dot ticks bypass stances/armor (frozen at cast); **(judge)** flat *permissive* component schema (not a discriminated union ‚Äî Haiku reliability), server normalizes/clamps/caps/drops. Deferred: stun (A.2), resource (A.3), blind/`hit_chance` + reliability + modes (P1), entities (P3). Full plan in the approved plan file; supersedes GAME_MECHANICS ¬ß3.
- 2026-07-03 ‚Äî **Turn model ‚Üí alternating (Worms-style), from simultaneous** (playtest feedback: clarity). `resolve(state,p1,p2)` ‚Üí `resolve_turn(state,action)`; simultaneous machinery deleted (speed-order, snapshot-delta, double-KO tiebreak, defense-priority tier); HP floors at 0. Effect/cooldown upkeep = end-of-owner-turn, install-after-tick. **Defenses are persistent stances** (frozen at cast, consume-on-hit, expire next turn). Round cap at round boundary. Events enriched (`target`+`effect`) for client-side result narration; status chips made readable. Judge untouched. Seeded starting-player + probabilistic evade deferred to P1.
- 2026-07-02 ‚Äî **Major direction change:** move from the closed 5-category set to an open-ended **generic effect grammar** (flavor infinite, mechanics = a small set of composable dimensions the LLM projects prompts onto), plus two **modes** (competitive / sandbox) and a **reliability (miss/fizzle‚Üípartial‚Üífull‚Üíoverload, backfire on overreach) system** driven mostly by counterplay with informed odds shown pre-Confirm and seeded RNG for replay. Full design in [[DESIGN.md]]; phased rollout in DESIGN ¬ß7. Comeback/rubber-band knob deferred. Supersedes SPEC ¬ß1 non-goals + GAME_MECHANICS ¬ß3/¬ß11.

## 2026-07-03 ‚Äî Bundle-cost retune (Batch B, step 1)
